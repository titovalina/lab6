# Задание 1.
## 1. @Invoke. 
### Текст задачи
Разработайте аннотацию @Invoke, со следующими характеристиками:
* Целью может быть только МЕТОД
* Доступна во время исполнения программы
* Не имеет свойств

Создайте класс, содержащий несколько методов, и проаннотируйте хотя бы один из них аннотацией @Invoke.
Реализуйте обработчик (через Reflection API), который находит методы, отмеченные аннотацией @Invoke, и вызывает их автоматически.

<img width="438" height="113" alt="image" src="https://github.com/user-attachments/assets/4872c1a3-6443-464e-9251-da4bb99931fe" />

### Алгоритм решения
1. Создать аннотацию @Invoke, применяемую только к методам и доступную во время выполнения.
2. Создать класс с несколькими методами и пометить один или несколько из них @Invoke.
3. С помощью Reflection API получить все методы класса.
4. Найти методы, помеченные @Invoke, и автоматически вызвать их.

## 2. @Default. 
### Текст задачи
Разработайте аннотацию @Default, со следующими характеристиками:
* Целью может быть ТИП или ПОЛЕ
* Доступна во время исполнения программы
* Имеет обязательное свойство value типа Class

Проаннотируйте какой-либо класс данной аннотацией, указав тип по умолчанию.
Напишите обработчик, который выводит имя указанного класса по умолчанию.

### Алгоритм решения
1. Создать аннотацию @Default, применяемую к классу или полю, с обязательным параметром типа Class.
2. Пометить класс аннотацией @Default, указав тип по умолчанию.
3. Реализовать обработчик, который считывает аннотацию и выводит имя указанного класса.

## 3. @ToString.
### Текст задачи
Разработайте аннотацию @ToString, со следующими характеристиками:
* Целью может быть ТИП или ПОЛЕ
* Доступна во время исполнения программы
* Имеет необязательное свойство valuec двумя вариантами значений: YES или NO
* Значение свойства по умолчанию: YES

Проаннотируйте класс аннотацией @ToString, а одно из полей – с @ToString(Mode.NO).
Создайте метод, который формирует строковое представление объекта, учитывая только те поля, где @ToString имеет значение YES.

<img width="418" height="80" alt="image" src="https://github.com/user-attachments/assets/7cd92d84-31ba-4dd2-8ca4-a15da9aee674" />

### Алгоритм решения
1. Создать аннотацию @ToString, применяемую к классу и полю, с параметром режима YES/NO.
2. Пометить класс аннотацией @ToString, а одно из полей — режимом NO.
3. С использованием Reflection API получить все поля класса.
4. Сформировать строковое представление объекта, включая только разрешённые поля.
